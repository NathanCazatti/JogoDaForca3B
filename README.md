# JogoDaForca3B
README - O que mudou no Jogo da Forca
Esta versão traz ajustes simples que tornam o jogo mais completo, estável e fácil de manter. Veja o que foi atualizado:
Palavras organizadas por temas e com dicas
As palavras não ficam mais todas misturadas. Elas foram separadas em categorias (programação, escola, jogos e filmes). O jogo sorteia um tema e mostra automaticamente uma dica antes de começar, facilitando a descoberta.
Opção de jogar novamente sem reiniciar
O código original fechava após uma partida. Agora, o programa pergunta se o jogador quer continuar, aceitando apenas "s" ou "n" e reiniciando tudo de forma limpa e segura.
Como executar
Abra o terminal ou prompt de comando, acesse a pasta do arquivo e digite: python jogodaforca.py. Requer Python 3.6 ou superior.
Próximas melhorias sugeridas:
*Criar um placar que salve as pontuações.
*Desenhar a forca na tela a cada erro.
*Adicionar níveis de dificuldade (fácil, médio, difícil).
*Tratar corretamente letras com acento e cedilha.
