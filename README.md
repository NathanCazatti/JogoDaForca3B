# JogoDaForca3B
Jogo da Forca - Python
Jogo educativo desenvolvido em Python para fins didáticos na disciplina de Programação no Desenvolvimento de Sistemas.

Como executar:
Certifique-se de ter o Python 3 instalado.
Abra o terminal ou prompt de comando na pasta do arquivo.

Execute:
em bash
|python jogo_da_forca3b_nathan.py|

Regras do jogo:
O programa escolhe uma palavra aleatória da lista interna.
O jogador tenta adivinhar a palavra digitando uma letra por vez.
Inicia com 6 vidas.
Letra correta: +10 pontos.
Letra errada: -1 vida e -2 pontos.
O jogo termina ao acertar a palavra ou zerar as vidas.
Ao final, o sistema pergunta se deseja jogar novamente.

Conceitos trabalhados:
Estruturas de repetição (while, for)
Listas e manipulação de strings
Criação e chamada de funções
Validação de entrada do usuário
Controle de fluxo e lógica condicional
